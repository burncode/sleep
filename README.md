# sleep
早睡签到打卡小程序

## 功能
* 1.自定义签到时间段
* 2.用户可以通过日历查看签到情况，也可以列表查看
* 3.用户每天签到的同时，可分享一些文字并发布到发现页，用户相互之间可以点赞
* 4.意见反馈（现在小程序推出这个功能了，当时还没有...）
